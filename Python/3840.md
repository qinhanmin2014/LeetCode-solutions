# 3840. House Robber V

## Solution 1 (time O(n), space O(n))

```python
# Dynamic programming
class Solution:
    def rob(self, nums: List[int], colors: List[int]) -> int:
        n = len(nums)
        dp0 = [0] * n
        dp1 = [0] * n
        dp1[0] = nums[0]
        for i in range(1, n):
            dp0[i] = max(dp0[i - 1], dp1[i - 1])
            dp1[i] = dp0[i - 1] + nums[i]
            if colors[i] != colors[i - 1]:
                dp1[i] = max(dp1[i], dp1[i - 1] + nums[i])
        return max(max(dp0), max(dp1))
```
