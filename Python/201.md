# 201. Bitwise AND of Numbers Range

## Solution 1 (time O(log(right)), space O(1))

```python
class Solution:
    def rangeBitwiseAnd(self, left: int, right: int) -> int:
        shift = 0
        while left < right:
            left = left >> 1
            right = right >> 1
            shift += 1
        return left << shift
```
