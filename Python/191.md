# 191. Number of 1 Bits

## Solution 1 (time O(log(n)), space O(1))

```python
class Solution(object):
    def hammingWeight(self, n):
        """
        :type n: int
        :rtype: int
        """
        ans = 0
        while n != 0:
            ans += n % 2
            n = n // 2
        return ans
```
