# 901. Online Stock Span

## Solution 1 (time O(n), space O(n))

```python
class StockSpanner(object):

    def __init__(self):
        self.idx = -1
        self.stack = [(-1, float("inf"))]

    def next(self, price):
        """
        :type price: int
        :rtype: int
        """
        self.idx += 1
        while price > self.stack[-1][1]:
            self.stack.pop()
        self.stack.append((self.idx, price))
        return self.idx - self.stack[-2][0]
```
