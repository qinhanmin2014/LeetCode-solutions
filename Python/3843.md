# 3843. First Element with Unique Frequency

## Solution 1 (time O(n), space O(n))

```python
class Solution:
    def firstUniqueFreq(self, nums: List[int]) -> int:
        d1 = collections.Counter(nums)
        d2 = collections.defaultdict(list)
        for k, v in d1.items():
            d2[v].append(k)
        ans_set = []
        for k, v in d2.items():
            if len(v) == 1:
                ans_set.append(v[0])
        ans_set = set(ans_set)
        if not ans_set:
            return -1
        for i in range(len(nums)):
            if nums[i] in ans_set:
                return nums[i]
```
