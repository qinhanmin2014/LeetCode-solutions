# 3838. Weighted Word Mapping

## Solution 1 (time O(nL), space O(1))

```python
class Solution:
    def mapWordWeights(self, words: List[str], weights: List[int]) -> str:
        ans = ""
        for word in words:
            ans += chr(ord('z') - sum([weights[ord(c) - ord('a')] for c in word]) % 26)
        return ans
```
