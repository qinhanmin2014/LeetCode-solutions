# 3839. Number of Prefix Connected Groups

## Solution 1 (time O(nk), space O(nk))

```python
class Solution:
    def prefixConnected(self, words: List[str], k: int) -> int:
        d = {}
        for word in words:
            if len(word) < k:
                continue
            if word[:k] in d:
                d[word[:k]] += 1
            else:
                d[word[:k]] = 1
        ans = 0
        for k, v in d.items():
            if v > 1:
                ans += 1
        return ans
```
