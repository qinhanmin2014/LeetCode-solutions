# 3824. Minimum K to Reduce Array Within Limit

## Solution 1 (time O(nlog(n)), space O(1))

```python
# Binary search
class Solution:
    def minimumK(self, nums: List[int]) -> int:
        left, right = 1, max(len(nums), max(nums))
        while left < right:
            mid = (left + right) // 2
            if sum([(x + mid - 1) // mid for x in nums]) <= mid * mid:
                right = mid
            else:
                left = mid + 1
        return right
```
