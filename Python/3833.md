# 3833. Count Dominant Indices

## Solution 1 (time O(n), space O(1))

```python
class Solution:
    def dominantIndices(self, nums: List[int]) -> int:
        n = len(nums)
        cur_sum = nums[n - 1]
        ans = 0
        for i in range(n - 2, -1, -1):
            if nums[i] * (n - i - 1) > cur_sum:
                ans += 1
            cur_sum += nums[i]
        return ans
```
