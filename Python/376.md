# 376. Wiggle Subsequence

## Solution 1 (time O(n), space O(1))

```python
# Dynamic programming
class Solution(object):
    def wiggleMaxLength(self, nums):
        """
        :type nums: List[int]
        :rtype: int
        """
        n = len(nums)
        if n < 2:
            return n
        up, down = 1, 1
        for i in range(1, n):
            if nums[i] > nums[i - 1]:
                up = max(up, down + 1)
            elif nums[i] < nums[i - 1]:
                down = max(down, up + 1)
        return max(up, down)

```

## Solution 1 (time O(n^2), space O(n))

```python
# Dynamic programming
class Solution(object):
    def wiggleMaxLength(self, nums):
        """
        :type nums: List[int]
        :rtype: int
        """
        if len(nums) == 0:
            return 0
        dp1 = [1] * len(nums)
        dp2 = [1] * len(nums)
        for i in range(1, len(nums)):
            for j in range(i):
                if nums[i] > nums[j]:
                    dp2[i] = max(dp2[i], dp1[j] + 1)
                elif nums[i] < nums[j]:
                    dp1[i] = max(dp1[i], dp2[j] + 1)
        return max(max(dp1), max(dp2))
```
