# 3842. Toggle Light Bulbs

## Solution 1 (time O(n), space O(n))

```python
class Solution:
    def toggleLightBulbs(self, bulbs: list[int]) -> list[int]:
        bulbs_state = [0] * 100
        for t in bulbs:
            if bulbs_state[t - 1] == 0:
                bulbs_state[t - 1] = 1
            else:
                bulbs_state[t - 1] = 0
        ans = []
        for i in range(len(bulbs_state)):
            if bulbs_state[i] == 1:
                ans.append(i + 1)
        return ans
```
