# 3834. Merge Adjacent Equal Elements

## Solution 1 (time O(n), space O(1))

```python
class Solution:
    def mergeAdjacent(self, nums: List[int]) -> List[int]:
        n = len(nums)
        ans = []
        for num in nums:
            cur_num = num
            while len(ans) > 0 and cur_num == ans[-1]:
                ans.pop()
                cur_num *= 2
            ans.append(cur_num)
        return ans
```
